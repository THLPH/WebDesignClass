The "Hotlink" Challenge

Problem: Pixiv servers return a 403 Forbidden error if the request originates from outside their domain.
Solution: We must inject a declarative rule: 


request -> i.pximg.net 
  |_ SET Header: Referer = "https://www.pixiv.net/"

This technique — known as "Referer Spoofing" — validates the extension as a legitimate browser agent.
